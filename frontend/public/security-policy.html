Information Security Policy

How Silk Route Logistics protects shipper and carrier data, and the security capabilities behind the platform.

Document SRL-ISP-2026-001  ·  Version 2.0  ·  Effective May 3, 2026  ·  Classification: Public

1. Our Security Commitment

Silk Route Logistics protects shipper and carrier data with industry-standard controls: encryption at rest and in transit, multi-factor authentication, role-based access, and continuous monitoring. The detailed implementation of these controls is maintained in internal documentation and is available under NDA to prospects who require it for procurement security review.

2. Capabilities

The SRL platform implements the following security capabilities:

  • Multi-factor authentication for all user accounts
  • Encrypted data storage for sensitive fields
  • TLS 1.2+ encrypted transport for all platform traffic
  • Role-based access control with least-privilege defaults
  • Audit logging of access events and data modifications
  • Continuous vulnerability monitoring
  • Automated daily backups with documented recovery objectives
  • Severity-based incident response with documented procedures

Each user account is provisioned with the minimum permissions required for their function. Detailed role definitions are available under NDA.

3. Regulatory Compliance

Silk Route Logistics operates in compliance with applicable regulations:

  • FMCSA Regulations: Carrier verification, safety scores, and authority validation
  • CCPA / State Privacy Laws: User data rights, deletion requests, privacy notices
  • PCI Awareness: No credit card data is stored directly; payment processing is handled through third-party providers
  • CAN-SPAM Act: Email communications include unsubscribe options

The Company maintains a Privacy Policy and Terms of Service that outline data handling practices and user rights.

4. Detailed Documentation Available on Request

For procurement teams requiring detailed security review: SRL maintains comprehensive internal security documentation covering authentication mechanics, encryption implementation, vendor due diligence, incident response procedures, and audit controls. The detailed document is available under NDA. To request a copy, contact compliance@silkroutelogistics.ai.

5. Contact

  • Security inquiries: compliance@silkroutelogistics.ai
  • Privacy-specific concerns: compliance@silkroutelogistics.ai
